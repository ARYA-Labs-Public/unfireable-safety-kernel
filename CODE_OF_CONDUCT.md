# Code of Conduct

This project follows the **Contributor Covenant v2.1** code of conduct. We are
committed to providing a welcoming, harassment-free experience for everyone
who participates in this community, regardless of background or identity.

Participants are expected to behave respectfully, accept constructive
feedback, and assume good faith in their interactions with maintainers and
other contributors. Behavior that harasses, demeans, or excludes others is
not tolerated in any project space (issues, pull requests, discussions,
chat, or events).

The full, authoritative text of the code we adopt is published here:

<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

## Reporting

Report violations to **conduct@aryalabs.io**. Reports are handled
confidentially. Maintainers will review every report and respond as quickly
as is practical.
